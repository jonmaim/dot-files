# dot-files

Useful for Ubuntu and Vim

Hack and Enjoy!

## Install

    cd ~;
    git clone git://github.com/jonmaim/dot-files.git dot-files;
    ./dot-files/install.sh;

## Vim Cheatcodes

* JavaScript indent guides `<leader>ig`
* Add new line and indent while in insert mode `<C-c>`
* Snippets `<C-j>`
